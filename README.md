# Home Assistant Add-on Repository: OmniRoute

A Home Assistant add-on repository that packages [OmniRoute](https://github.com/diegosouzapw/OmniRoute) — a free open-source AI gateway with 160+ providers, smart auto-fallback, OpenAI-compatible API, and built-in dashboard.

## Add this repository

In Home Assistant, go to **Settings → Add-ons → Add-on Store → ⋮ (top right) → Repositories** and paste:

```
https://github.com/ad/ha-omnirouter
```

Then install the **OmniRoute** add-on from the store.

## Add-ons

- [OmniRoute](./omnirouter) — AI gateway, dashboard, OpenAI-compatible API.

## Supported architectures

- `amd64`
- `aarch64`

(Upstream Docker image is published only for these two architectures.)
